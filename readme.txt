== Description ==

Add some contact information about your company or product in after header inner.
This plugin requires the [OceanWP](https://oceanwp.org/) theme to be installed.

== Installation ==

= USING WORDPRESS PLUGIN INSTALLER =

1. Go to your WordPress Dashboard, 'Plugins > Add New'.
2. Search for 'Ocean Contact Info'.
3. Click 'Install' and then 'Activate'.
4. Done!

= MANUAL INSTALLATION =

1. Download the 'ocean-contact-info' zip file.
2. Extract the content and copy to the `/wp-content/plugins/` directory of your WordPress installation.
3. Navigate to your WordPress dashboard, 'Plugins > Installed Plugins'.
4. Find the 'Ocean Contact Info' plugin and activate.
5. Done!

== Frequently Asked Questions ==

= I installed the plugin but it does not work =

This plugin will only function with the [OceanWP](https://oceanwp.org/) theme.

= Is Ocean Contact Info free? =

Ocean Contact Info is a 100% free WordPress plugin without any limitations on its features.
